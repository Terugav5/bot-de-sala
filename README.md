# bot-de-sala


Esse bot de sala funciona no ld player aonde as configs sao personalizadas 1604x 720y dpi 360 fizer isso pode dar comando .cs (senha sala ) se nao tiver a senha vai sem a senha

This room bot works on LD Player where the settings are customized: 1604x 720y dpi 360. If you do this, you can run the command `.cs (room password)`. If you don't have the password, it will work without it.
